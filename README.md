# BlockDrop

BlockDrop ist ein kleines Falling-Blocks-Spiel als statische Web-App. Fallende Blockformen werden in einem Raster bewegt und rotiert. Vollstaendige horizontale Reihen verschwinden und bringen Punkte. Die Runde endet, wenn neue Formen nicht mehr in das Spielfeld passen.

## Spielmodi

### Normalmodus

Im Normalmodus spielst du endlos, bis Game Over erreicht ist. Alle 10 geloeschten Linien steigt das Level, und die Blockformen fallen schneller. Der beste Punktestand wird separat als Normalmodus-Highscore im Browser gespeichert.

### Challenge-Modus

Im Challenge-Modus loest du mehrere Level nacheinander. Jedes Level hat ein klares Ziel, zum Beispiel eine bestimmte Punktzahl oder eine Anzahl geloeschter Linien. Manche Level starten mit neutralen Hindernis-Bloecken im Spielfeld. Nach einem geschafften Level wird das Board zurueckgesetzt und das naechste Ziel startet. Der beste Challenge-Gesamtstand wird separat gespeichert.

## Steuerung

- Pfeil links/rechts: Block bewegen
- Pfeil unten: schneller fallen
- Pfeil oben oder Leertaste: rotieren
- P: Pause ein/aus
- R: Neustart

## Mobile Steuerung

Auf Touch-Geraeten nutzt du die Buttons unter dem Spielfeld:

- Links
- Rechts
- Rotieren
- Runter
- Drop

Links, Rechts und Runter wiederholen beim Halten. Das Mobile-Layout ist fuer Hochformat optimiert, damit Board, wichtige Werte, Next-Piece und Touch-Buttons zusammen sichtbar bleiben.
